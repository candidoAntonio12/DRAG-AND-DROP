# DRAG AND DROP

Este repositorio é um registro do meu aprendizado sobre a API nativa javascript drag and drop
focado em enteder como os dados são transferidos no dom para isso acontecer temos que possuir
2 elementos um que sera o dragzone que a area que são os elementos que serão transferidos de uma área
para outra e outro sera o dropzone que a area onde os elementos serão soltos aprende sobre os 
seguintes termos e objectos e funções:

draggable - este atributo vai ser dado ao elemento que queremos arrastar é o gatilho para ele ser pego pelo mouse
dataTransfer - o Objecto que sera a maleta ou onde colocaremos os dados
preventDefault - esta é a função que vai permitir que o navegador não accione a sua função padrão que é bloquear o drop.
